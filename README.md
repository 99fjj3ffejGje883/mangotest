George jsDelivr export

This repo is meant to be served from GitHub.

Playable HTML link:

https://rawcdn.githack.com/99fjj3ffejGje883/george/main/index.html

SVG wrapper link:

https://cdn.jsdelivr.net/gh/99fjj3ffejGje883/george@main/index.svg

jsDelivr serves HTML as text/plain, so index.html shows source code there.
The index.svg file redirects to the playable Githack HTML link and includes a
visible Play Game button as a fallback.

Keep this repo public. CDN links cannot serve private GitHub repos.
